## 2.3 for 是 Go 的 “while”（For is Go's "while"）

示例：for-is-gos-while.go

  package main

  import "fmt"

  func main() {
    sum := 1
    for sum < 1000 {
      sum += sum
    }
    fmt.Println(sum)
  }

基于此可以省略分号：C 的 while 在 Go 中叫做 for 。
