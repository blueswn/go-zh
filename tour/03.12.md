## 03.12 nil 切片（Nil slices）

示例：nil-slices.go

    package main

    import "fmt"

    func main() {
    	var s []int
    	fmt.Println(s, len(s), cap(s))
    	if s == nil {
    		fmt.Println("nil!")
    	}
    }

切片的零值是 nil 。

nil 切片的长度和容量为 0 且没有底层数组。
