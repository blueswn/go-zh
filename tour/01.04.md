## 1.4函数（Functions）

示例：functions.go

	package main

	import "fmt"

	func add(x int, y int) int {
		return x + y
	}

	func main() {
		fmt.Println(add(42, 13))
	}

函数可以没有参数或接受多个参数。

在这个例子中， add 接受两个 int 类型的参数。

注意类型在变量名 之后 。

（参考 [这篇关于 Go 语法定义的文章](https://blog.go-zh.org/gos-declaration-syntax)了解类型以这种形式出现的原因。）
