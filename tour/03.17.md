## 03.17 range（续）（Range continued）

示例：range-continued.go

    package main

    import "fmt"

    func main() {
    	pow := make([]int, 10)
    	for i := range pow {
    		pow[i] = 1 << uint(i) // == 2**i
    	}
    	for _, value := range pow {
    		fmt.Printf("%d\n", value)
    	}
    }

可以将下标或值赋予 _ 来忽略它。

若你只需要索引值，去掉 , value 的部分即可。
