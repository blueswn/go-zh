## 3.02 结构体（Structs）

示例：structs.go

    package main

    import "fmt"

    type Vertex struct {
    	X int
    	Y int
    }

    func main() {
    	fmt.Println(Vertex{1, 2})
    }

一个结构体（ struct ）就是一个字段的集合。

（而 type 声明就是定义类型的。）
