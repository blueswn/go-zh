## 2.2 for（续）（For continued）

示例：for-continued.go

    package main

    import "fmt"

    func main() {
      sum := 1
      for ; sum < 1000; {
        sum += sum
      }
      fmt.Println(sum)
    }

循环初始化语句和后置语句都是可选的。
