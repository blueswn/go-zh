## 3.03 结构体字段（Struct Fields）

示例：struct-fields.go

    package main

    import "fmt"

    type Vertex struct {
    	X int
    	Y int
    }

    func main() {
    	v := Vertex{1, 2}
    	v.X = 4
    	fmt.Println(v.X)
    }

结构体字段使用点号来访问。
