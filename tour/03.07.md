## 03.07 切片（Slices）

示例：slices.go

    package main

    import "fmt"

    func main() {
    	primes := [6]int{2, 3, 5, 7, 11, 13}

    	var s []int = primes[1:4]
    	fmt.Println(s)

    	s[2] = 4
    	fmt.Println(s)
    }

每个数组的大小都是固定的。 而切片则为数组元素提供动态大小的、灵活的视角。 在实践中，切片比数组更常用。

类型 []T 表示一个元素类型为 T 的切片。

以下表达式为数组 a 的前五个元素创建了一个切片。

    a[0:5]
